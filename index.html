参数面板

参数符号速查 K、H、P、c 价 等 · 小屏可展开

速查：K 为总结周期（每 K 轮触发摘要）；H 为窗口内参与保留的轮数上界；P / A 为系统提示与单轮输出 token 数；c_in c_cache c_out 为每百万 token 的输入/缓存/输出价（元）。

策略
两种可切换
核心参数

P：首条系统消息 token 数（系统提示）。

A：模型每轮回复的 token 数。

N_max：横轴上最多模拟的轮数 N。

K：每隔 K 轮触发生成或更新摘要（总结周期）。

H：窗口内参与保留的对话轮数上界（与策略 B 有关）。

操作
API 价格

c_in：每百万 token 的首次非缓存输入价。

c_cache：每百万 token 的缓存读入单价。

c_out：每百万 token 的生成输出价。

摘要与压缩
压缩比 R / 直接 S

R：压缩比，与摘要长大致满足 S = K×A/R。

S：压缩后摘要的 token 数（直接设 S 时）。
